# How to run:
- open 2 command shells
  - on the first one:
    - navigate to packages/simulation-server
    - `npm i`
    - `npm run start`
  - on the second one:
    - navigate to packages/simulation-visual
    - `npm i`
    - `npm run start`

# Theh simulator

The simulation simulates automated-farming actors i.e. watering robot and water pumps.
